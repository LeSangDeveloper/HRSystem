# HRSystem

A simple **Human Resource Management System** built with **ASP.NET Core 3.1**.

This project is a basic HR web application created for learning and demonstration purposes.

## Features

* ASP.NET Core 3.1
* Razor Pages
* MVC-style project structure
* Configurable via `appsettings.json`

## Technologies

* ASP.NET Core 3.1
* C#
* HTML, CSS, JavaScript

## Getting Started

### Prerequisites

* .NET Core SDK 3.1
* Visual Studio 2019 / Visual Studio Code

### Clone the Repository

```bash
git clone https://github.com/LeSangDeveloper/HRSystem.git
```

### Run the Application

```bash
cd HRSystem
dotnet restore
dotnet run
```

### Access the Application

Open your browser and navigate to:

```
https://localhost:5001
```

(or the URL shown in the terminal)

## Project Structure

```
/Models      # Data models
/Pages       # Razor Pages
/wwwroot     # Static files
Program.cs   # Application entry point
Startup.cs   # Application configuration
```

## Notes

* This project is intended for learning purposes.
* You can extend it with authentication and database integration.
